## Issue: Introducing DebateHeat

**Lead**

If you orchestrate multiple AI personas on a single topic, you already know the bottleneck: **aggregation**. DebateHeat is an open-source dashboard that turns each day’s run into **one scannable page**—not another PDF transcript.

**What it shows**

- Top ideas with composite scores  
- Heatmaps for ideas × criteria, keywords, and agent groups  
- Radar chart for leading options  
- Short bullets where “VC,” “Ethics,” and “Engineering” views diverge  
- Optional deltas vs yesterday  

**How it fits your stack**

The app reads **one JSON file per date**. Your debate job (Python, Node, whatever) writes that file; DebateHeat handles the visuals. A mock generator is included so you can try the UI first.
